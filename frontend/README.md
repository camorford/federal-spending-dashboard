# Frontend

React 18 dashboard with Vite and Tailwind CSS.

## Development
```
npm run dev
```

Runs on http://localhost:5173

## Key Files

- `src/pages/` - Dashboard, Awards, Agencies, Recipients, Search
- `src/components/` - Layout, StatCard, SyncPanel
- `src/services/api.js` - Backend API client

## Build
```
npm run build
```
